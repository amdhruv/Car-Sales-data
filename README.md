# 🚗 Car Sales Dashboard (Power BI)

## 📊 Overview
This project presents an interactive Car Sales Dashboard built using Power BI to analyze sales performance, customer trends, and revenue insights. The dashboard helps in understanding key business metrics and supports data-driven decision-making.

---

## 📁 Dataset
- Source: Excel file (Car Sales.xlsx)
- Includes:
  - Car details (Model, Brand, Category)
  - Sales data (Units Sold, Revenue)
  - Customer information
  - Date fields for trend analysis

---

## 🛠️ Tools & Technologies
- Power BI (Dashboard & Visualization)
- Microsoft Excel (Data Source)
- DAX (Calculations & Measures)
- Power Query (Data Cleaning)

---

## ⚙️ Project Workflow
1. Data Loading  
   - Imported dataset from Excel into Power BI  

2. Data Cleaning  
   - Handled missing values  
   - Removed duplicates  
   - Formatted columns  

3. Data Transformation  
   - Used Power Query for preprocessing  

4. Data Modeling  
   - Created measures using DAX  
   - Example:
     - Total Sales  
     - Profit  
     - Sales Growth  

5. Dashboard Creation  
   - Built interactive visuals and KPIs  

---

## 📈 Dashboard Features
- Total Revenue Analysis  
- Top-Selling Car Models  
- Sales Trend Over Time  
- Region-wise Sales Distribution  
- Category-wise Performance  
- Interactive Filters & Slicers
